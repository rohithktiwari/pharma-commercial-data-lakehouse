# Business Rules

The following business rules were applied during data processing.

- Remove duplicate records.
- Handle missing values.
- Standardize date formats.
- Standardize column names.
- Validate primary and foreign key relationships.
- Remove invalid transactions.
- Create business-ready fact tables.
- Create dimension tables.
- Generate commercial KPIs.
- Store processed datasets using Delta Lake.
- Organize data using the Medallion Architecture.
- Govern analytical tables through Unity Catalog.
